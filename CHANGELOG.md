# 更新日志

## v2.0.1 (2026-06-20)

### 修复
- 修复大学语文部分题目选项显示不全的问题（选项 A/B 被错误合并到题目正文中）

## v2.0.0 (2026-06-20)

### 多学科平台重构
- 重构为多学科通用平台，支持通过 `subjects.json` 配置多门学科
- 新增学科选择器，可在不同题库间切换
- 页面标题动态显示当前学科名称
- 学科支持 `author`、`university`、`profession` 元数据字段

### 新题库
- 新增《大学语文各课测试题》（9章124题），覆盖世说新语、庄子、论语等课文
- 包含题型：单选、多选、填空、判断、简答、古文今译

### 新功能
- 贡献题库弹窗，含联系邮箱和微信
- 考试模式题型数量和分值完全可自定义
- 标题区域显示大学/专业信息

### 改进
- 题库数据迁移至 `subjects/` 目录，结构更清晰
- README 增加多学科添加说明和贡献指南
- 页面移除作者显示，保持低调

## v1.0.0 (2026-06-19)

### 初始版本
- 三种学习模式：考试（按章节比例抽题）、背题、逐章测试
- 错题本功能（localStorage 持久化）
- 考试计时、自动交卷、成绩记录
- 题型分值自定义
- 题目反馈链接（GitHub Issues）
- 内置《中华民族共同体概论》题库（1550题/16章）
- 纯静态、零依赖、GitHub Pages 一键部署
